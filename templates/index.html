Help & Features Guide

  • Data Source — All data is loaded from Snowflake on startup.
  • All changes are deferred — Edits to any field (recommendation, DEC fields, memo, process, BA type flags) are held in memory until Save is clicked.
  • Save Changes — Writes all pending changes to Snowflake in a single batch. The badge shows how many records have unsaved changes.
  • Refresh () — Reloads data from Snowflake. Unsaved changes will be lost.
  • Import — CSV/Excel files can still be imported for Canvas ID matching (JIB/Rev/Vendor flags).
  • Dev Notes () — Opens the development notes document in Word.

  • Recommendation filter — Multi-select dropdown to filter by one or more recommendation types. No checkboxes checked = show all.
  • SSN Match — Filter by Yes (100), Partial (1-99), or No (0).
  • Name / Address score — Set min and max thresholds for Name Score and Address Score.
  • Search box — Type to search across all visible columns (server-side, case-insensitive).
  • Recommendation cards — Click any recommendation card to filter to that type. Resets page to first page.
  • Clear — Resets all filters at once and clears row selections.

  • Column sorting — Click any column header to sort ascending/descending (server-side).
  • Column reordering — Drag column headers to rearrange the table layout. The checkbox column (left) is fixed in place.
  • Column resizing — Drag the right edge of any column header to adjust width. All data columns are resizable (SSN, scores, Status, Process, Canvas/DEC name/address/CSZ, Canvas ID, DEC Code). Minimum width is 50px.
  • Show/Hide Columns — Dropdown to toggle visibility of any column. Columns are color-coded: Canvas DEC General
  • Hidden by default — UID, Address Lookup, Canvas ID, DEC Code, JIB, Rev, Vendor, and Actions columns are hidden initially. Use Show/Hide Columns to reveal them.
  • Sticky headers — Column headers stay visible when scrolling through table rows.

  • Row checkboxes — Select individual rows; selection persists across pages.
  • Select All — Header checkbox selects/deselects all visible rows on the current page.
  • Cell click — Click a cell to highlight it (blue). The cell text is ready to copy.
  • Shift+click (same column) — Select a range of cells in that column from the last-clicked cell to the current one.
  • Shift+click (different column) — Select the entire row of the clicked cell; text is auto-selected for native copy.
  • Ctrl+C — Copy all highlighted cell values to clipboard (one value per line).

  • Edit button () — Opens a side-by-side modal showing Canvas (read-only) vs DEC (editable) fields with match scores. Edit DEC name, address, city, state, zip, and recommendation.
  • Quick Approve () — Instantly sets a single record's recommendation to APPROVED without opening the edit modal.
  • Approve Selected — Bulk-approves all checkbox-selected rows in one action.
  • Process column — Click any Process cell to open a dropdown with options: "Add new BA and address", "Merge BA - add addr", "Merge BA and address", or "Manual Review -DNP". Defaults are auto-populated based on the recommendation status.
  • Process right-click — Right-click the Process column header to set all visible rows to one option at once. Changes are saved immediately via deferred save.
  • JIB / Rev / Vendor checkboxes — Toggle BA type flags directly in the table. If multiple cells are selected in the column, the toggle applies to all selected rows at once.
  • Memo column — Click any Memo cell to type a free-text note. Saves automatically on Enter or when you click away. Press Escape to cancel. Also editable in the Edit modal.
  • Save Changes — Persists all pending changes (recommendations, DEC fields, memo, process, BA type flags) to Snowflake in one batch. The badge shows how many records have unsaved changes.

  • Import Canvas IDs — Select JIB, Rev, or Vendor from the BA type dropdown, then pick a CSV/TXT file containing Canvas IDs (one per line) to bulk-set that flag.
  • Export filtered — Downloads all currently filtered records as an Excel file (.xlsx) using the Export button in the toolbar.
  • Export selected () — Downloads only checkbox-selected rows as Excel (.xlsx).

  • Orange rows — Canvas name or address contains "trust", "trst", or "tr".
  • Red cells — Cell contains "DO NOT USE" (or variations: DON'T USE, DNU, etc.).
  • Score badges — 100 perfect, 90+ high, 75+ medium, <75 low.
  • Name score tooltip — Hover over a Name Score below 45 to see a tooltip explaining the low score may be because the name exists in the address field.
  • SSN badges — Yes = 100 (perfect match), No = 0 (no match).
  • Recommendation badges — ADD NEW BA no valid SSN or not found in DEC, MATCH BA + ADDRESS SSN + name + address confirmed, MATCH BA - REVIEW ADDRESS BA confirmed but address below threshold, NEEDS REVIEW SSN matches but name mismatch or trust/estate.
  • Cell selection highlight — Blue highlight on clicked/selected cells.
